SemanticServer (Console Edition)Ультракомпактный семантический сервер на Free Pascal (Lazarus), оптимизированный для работы на Edge-устройствах (например, Jetson Nano).🚀 Ключевые особенностиМинимализм: Исполняемый файл весит всего 3.5 Мб, включая полноценный HTTP-сервер и драйверы БД.Скорость: Ядро обхода графа (алгоритм «Эстафета хвоста») занимает всего 30 строк кода и работает в режиме Zero-Allocation (без лишних выделений памяти).Оптимизация под кэш L1: Использование целочисленных массивов вместо строковых списков для стека навигации.Многопоточность: Обработка запросов через Threaded := True с использованием SQLite в режиме WAL (Write-Ahead Logging).🏗 Структура проекта1. semanticserver.lpr (Ядро системы)Роль: Точка входа и сетевой диспетчер.Функции:Инициализация базы данных при старте.Маршрутизация (Routing): / — главная, /forum — запуск эстафеты.Управление жизненным циклом потоков-воркеров.2. unit_db.pas (Слой данных)Роль: Адаптер SQLite.Особенности:Настроенный busy_timeout (5000мс) для предотвращения блокировок в Linux.Атомарные запросы к таблицам nodes и chrono.3. unit_worker.pas (Интеллект системы)Роль: Реализация алгоритма «Эстафета хвоста».Механизмы:TailStack: array of Integer — высокоскоростной стек навигации.HTML_Acc: TStringBuilder — эффективное накопление HTML-кода без фрагментации памяти.ArtistDispatcher: Система разделения логики. Готовит данные для визуализации (режим Artist) отдельно от текстового представления (режим Viewer).🛠 Технический стекЯзык: Free Pascal (FPC)IDE: Lazarus 4.6.0 (Linux)База данных: SQLite 3Библиотеки: fphttpserver (стандартный сетевой стек FPC)📡 Планы на развитие (Завтра)Frontend Split: Разделение браузера на две зоны (Левая — текст, Правая — Canvas/GPU графика).Artist Engine: Внедрение JavaScript-рендерера, который будет превращать посылку FArtistBuffer в визуальное созвездие.Аналитика: Наполнение метода agDeepAnalysis для фонового пересчета весов семантических связей.Как запустить:fpc semanticserver.lpr (или Ctrl+F9 в Lazarus)../semanticserverОткрыть в браузере: http://localhost:8080/forum
