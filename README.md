SemanticServer (Console Edition)Ультракомпактный семантический сервер на Free Pascal (Lazarus), оптимизированный для работы на Edge-устройствах и Jetson Nano.Ключевые особенностиМинимализм: Исполняемый файл весит всего 3.5 Мб. Включает в себя HTTP-сервер и драйверы БД.Алгоритм «Эстафета хвоста»: Ядро обхода графа реализовано в 30 строках кода без использования рекурсии.Оптимизация памяти:Использование динамических массивов целых чисел (array of Integer) для стека навигации вместо медленных строковых списков.Использование TStringBuilder для формирования HTML, что исключает фрагментацию памяти при сборке больших страниц.Многопоточность: Обработка запросов в режиме Threaded := True. Работа с SQLite настроена в режиме WAL (Write-Ahead Logging).Архитектура проекта1. Главный модуль (semanticserver.lpr)Является точкой входа и сетевым диспетчером.Инициализирует объект базы данных при старте.Настраивает маршрутизацию (Routing): корень / и ветка /forum.Управляет созданием и уничтожением потоков-воркеров для каждого запроса.2. Модуль базы данных (unit_db.pas)Адаптер для работы с SQLite.Реализует атомарное чтение данных (ID, Контент, Хронология).Содержит настройки стабильности для Linux (PRAGMA busy_timeout).3. Модуль логики (unit_worker.pas)Сердце системы, где реализован алгоритм обхода.Навигатор: Основной цикл, использующий целочисленный массив как LIFO-стек.ArtistDispatcher: Метод-распределитель, который отделяет подготовку данных для визуализации (Artist) от формирования HTML (Viewer).RenderNodeHTML: Изолированная процедура формирования визуального представления карточек.Технический стекЯзык: Free Pascal (FPC)Сетевой стек: fphttpserverБаза данных: SQLite 3Платформа: Linux (x86_64, ARM64 / Jetson Nano)Ближайшие задачиFrontend Split: Разделение страницы браузера на две независимые зоны через CSS Grid.Artist Engine: Разработка JavaScript-части для отрисовки семантических созвездий на Canvas.Deep Analysis: Наполнение методов фоновой аналитики для пересчета весов узлов в режиме реального времени.Как запуститьСкомпилируйте проект в Lazarus (Ctrl+F9) или через fpc semanticserver.lpr.Убедитесь, что файл forum.db находится в папке с программой.Запустите бинарный файл: ./semanticserver.Откройте в браузере: http://localhost:8080/forum.
